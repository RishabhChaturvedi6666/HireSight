## ✨ Features

- 🔥 **Skill Demand Ranking** — Top 20 most in-demand skills
- 💰 **Salary Benchmarking** — Which skills pay the most
- 📍 **Location Intelligence** — Best cities for AI/DS jobs
- 📈 **Experience Analysis** — Salary vs years of experience
- 🏠 **Work Type Trends** — Remote vs Hybrid vs Onsite
- 🔍 **Interactive Filters** — Filter by experience, location, work type
